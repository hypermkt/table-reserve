# TableReserve

## Intro
TableReserve is a restaurant reservatation platform on Laravel PHP Framework.

## Installation

```sh
$ git cloen git@github.com:hypermkt/table-reserve.git
$ cd table-reserve
$ composer install
$ npm install
$ php artisan key:generate
$ php artisan migrate
# Fix database and Twitter key and secret info.
$ cp .env.example .env
$ php artisan serve
```
